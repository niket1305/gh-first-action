# gh-first-action

A minimal repository to demonstrate a first GitHub Action and CI workflow.

## About

This repository contains a simple example for setting up GitHub Actions. It is intended as a starting point for learning how to create workflows, run jobs, and automate common tasks.

## Contents

- `.github/workflows/` — GitHub Actions workflow definitions
- `src/` — example source code (if present)

## Getting started

1. Clone the repo:

   git clone https://github.com/niket1305/gh-first-action.git
   cd gh-first-action

2. Inspect workflows in `.github/workflows/` and modify them to fit your project.

## Usage

- Push commits to trigger workflows according to the definitions in `.github/workflows/`.
- Check the Actions tab on GitHub to see runs, logs, and artifacts.
